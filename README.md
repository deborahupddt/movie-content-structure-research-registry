# movie-content-structure-research-registry
该项目围绕在线电影与电视剧网站的内容结构展开系统整理，对首页模块分布方式、导航层级关系、播放页信息区块划分以及栏目更新节奏进行持续记录。通过结构化文字归档建立资料库，用于对比不同影视平台在页面组织与内容呈现策略方面的变化趋势。仓库仅保存公开页面资料，仅供信息整理与研究参考使用，不涉及影视资源文件或在线播放功能，确保资料归档与内容提供相互独立。
